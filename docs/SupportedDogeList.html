# <SupportedDogeList contracts selected setSelected blockExplorerUrl />

Component for selecting the supported Doge contract from a list.

Each entry will display the name of the name and symbol of the contract, the current balance of the contract, and an icon that provides a link to view the contract on the specified block explorer.

PropTypes:
Name Type Required Description Default
contracts arrayOf Yes An array of the supported Doge contracts.
selected string Yes The currently selected contract.
setSelected func Yes The function for setting the selected contract.
blockExplorerUrl string No The url for the block explorer. "https://explorer.harmony.one/#/address/"
