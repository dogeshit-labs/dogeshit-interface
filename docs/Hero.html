# <Hero imgName heroContent children />

A hero component. This will display the Hero image at the top of the page.

The heroContent will be rendered overtop of the Hero image. The children will be rendered directly beneath the Hero image.

PropTypes:
Name Type Required Description Default
imgName string No The name of the image to use as the Hero, should be accessible under /public/images/{sm,md,lg,xl,xxl,xxxl,xxxxl} ""
heroContent element No
children element Yes
