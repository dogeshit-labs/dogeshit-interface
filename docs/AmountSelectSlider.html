# <AmountSelectSlider maxAmount value setValue sliderUnits />

Component for selecting an amount. Selection can happen from either a number input or slider, which are both synced.

PropTypes:
Name Type Required Description Default
maxAmount number No The largest value that can be selected. 100
value number Yes The variable to access the selected value from. Often a state variable.
setValue func Yes The function for setting the selected variable. Often a setState method.
sliderUnits string No The string representing the units of the amount. ""
